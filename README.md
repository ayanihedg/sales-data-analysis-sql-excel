#  Sales Data Analysis (SQL & Excel)

##  Project Overview
This project analyzes a small sales dataset to extract meaningful insights using SQL and Excel.
It focuses on understanding sales performance by city, payment method, and total revenue.

##  Tools Used
- SQL (table creation, aggregation, grouping)
- Excel (data validation & analysis)

##  Dataset
The dataset contains:
- Client name
- City
- Product
- Quantity
- Unit price
- Payment method
- Sale date

##  Analysis Questions
- What is the total revenue generated?
- Which cities generate the highest sales?
- What payment methods are most used?
- How is sales distributed by city and payment type?

##  Key Insights
- Algiers generates the highest total sales revenue.
- Cash and card payments are the most common.
- Laptops (PC Portable) contribute significantly to total revenue.
